# 🏦 Sistema Bancário em Python (POO)

Projeto em Python que simula um sistema bancário usando Programação Orientada a Objetos. Permite criar clientes e contas, realizar depósitos, saques e consultar histórico de transações via menu interativo no terminal.

---

## 💡 Funcionalidades

- Criar clientes
- Criar contas bancárias
- Depósitos
- Saques com validação de saldo
- Consulta de saldo
- Histórico de transações
- Menu interativo no terminal

---

## 🧠 Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Herança
- Encapsulamento
- Polimorfismo
- Abstração
- Composição de classes

---

```bash
python sistema_bancario.py
